Make your website instantly more accessible! Focusable restores and enhances the visible focus ring for keyboard users, ensuring everyone can navigate your site with confidence.

== Description ==
Focusable is a lightweight, plug-and-play WordPress plugin that displays a beautiful, customizable focus ring around interactive elements as users navigate with the keyboard. Many themes remove or hide the default focus outline, making it difficult for keyboard and assistive technology users to know where they are on the page. Focusable solves this problem in seconds—no coding required.

**Key Benefits:**
- Instantly improves accessibility and meets WCAG guidelines.
- Helps users with disabilities, power users, and anyone who prefers keyboard navigation.
- Works with any theme, even if the default focus outline is removed.
- Customizable transition and appearance via plugin settings.

== Features ==
- Adds a visible focus ring to all focusable elements (links, buttons, form fields, etc.)
- Keyboard navigation support out of the box
- Customizable ring style and transition
- Lightweight and fast—no bloat
- Compatible with all major browsers and themes
- Developer-friendly: extend or style as needed

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Focusable"
3. **Activate** Focusable from your Plugins page
4. (Optional) Adjust settings in the admin area to customize the focus ring appearance.

== Frequently Asked Questions ==

= Who should use Focusable? =
Anyone who wants to make their website more accessible, especially for keyboard and screen reader users. It’s essential for site owners, agencies, and developers who care about usability and compliance.

= Will this work with my theme? =
Yes! Focusable is designed to work with any WordPress theme, even if the theme removes the default focus outline.

= Is it customizable? =
Yes, you can adjust the transition and appearance of the focus ring from the plugin settings.
